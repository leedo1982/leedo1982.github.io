# intro

S는 " (U) " 형식 을가집니다. 여기서 U는 올바르게 중첩 된 문자열입니다.
S는 " VW " 형식을 갖습니다. 여기서 V와 W는 올바르게 중첩 된 문자열입니다.

## Bic-O

## Assume 
N은 [ 0 .. 1,000,000 ] 범위의 정수
문자열 S는 " ( "및 / 또는 " ) " 문자
